P Powerball Number Picker

Frequency-based lottery number generator

Data Source

  1. Go to thelott.com/powerball/results
  2. Scroll down to the Statistics section
  3. Click the "Last Drawn" tab to see how many weeks ago each number was last drawn
  4. Select all the text in the Winning Numbers table (the number + weeks columns)
  5. Copy and paste it into the textarea below
Each line: <number> <weeks> week(s) — tolerant of extra spaces, tabs, and variations.
Pick the frequency .txt file — works with file:/// (no server needed).
